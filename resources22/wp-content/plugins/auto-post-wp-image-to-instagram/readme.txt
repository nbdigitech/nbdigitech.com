== Description ==
This plugin using for auto post image from Wordpress to Instagram. This plugin support jpg image only and you can choose post types these auto post image Instagram
If you want advanced custom please contact me. 
my skype: nhakhtn.
my email (and hangouts): nhakhtn@gmail.com

Note: we can only auto post jpg image - this plugin doesn't accept png or bmp image.
you can set which post types can be auto post image to Instagram in Settings.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/wp2instagram` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->wp2instagram screen to configure the plugin. Update Instagram username and password then submit.
4. Go to post management then add new post. Set Title and feature image then click publish. the Image will auto post to instagram with title.
5. you can set which post types can be auto post image to Instagram.
